# 教育视频知识图谱生成器

这是教育视频推荐系统的第一个模块，当前重点解决“根据用户输入词动态生成知识图谱”的问题。

当前版本已经不再读取预置领域图谱，而是根据用户输入的学习主题实时生成。输入 OpenAI API Key 后，系统会调用 LLM 生成结构化知识图谱；不输入 Key 时，只显示占位图，不伪造具体前置知识。

- 学习目标节点
- 具体前置知识节点
- 目标概念本身的核心节点
- 与目标相关的延伸节点
- 应用节点
- 节点之间的学习依赖关系

直接在浏览器中打开 `index.html` 即可运行原型。

## 当前生成方式

当前版本支持两种生成方式：

### 1. LLM 生成

```text
输入词语
  -> 调用 OpenAI Responses API
  -> 使用 JSON Schema 约束输出
  -> 得到具体 nodes 和 edges
  -> 前端校验节点和关系
  -> 输出可视化图谱和 JSON 数据
```

### 2. 无 API Key 时

如果没有输入 API Key，系统只会显示占位图，提示需要 LLM 分析。这样可以避免把“基础概念、核心定义、应用场景”这类固定模板误当成真实知识图谱。

注意：当前 GitHub Pages 只能托管静态页面，不能安全保存 API Key。仓库已加入 `server.js` 本地后端，由后端从 `OPENAI_API_KEY` 环境变量读取 Key 并调用 LLM。正式线上版应部署到 Vercel、Render 或其他支持环境变量的后端平台。

## 本地运行 LLM 生成

1. 复制 `.env.example` 为 `.env`。
2. 在 `.env` 中填写 `OPENAI_API_KEY`。
3. 运行 `npm start` 或 `node server.js`。
4. 打开 `http://localhost:3000`。

## 后续模块

1. 接入 LLM 或课程知识库，生成更具体的知识点。
2. 加入知识图谱质量校验，例如节点去重、关系合法性检查。
3. 再进入诊断问题生成模块。
4. 接入真实视频元数据和字幕文本。
5. 加入推荐评分：语义相似度、知识点匹配度、学习者水平。
