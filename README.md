# 教育视频知识图谱生成器

这是教育视频推荐系统的第一个模块，当前重点解决“根据用户输入词动态生成知识图谱”的问题。

当前版本已经不再读取预置领域图谱，而是根据用户输入的学习主题实时生成。输入 OpenAI API Key 后，系统会调用 LLM 生成结构化知识图谱；不输入 Key 时，会使用本地规则生成器作为备用方案。

- 学习目标节点
- 前置知识节点
- 核心定义节点
- 表示方法节点
- 关键性质节点
- 解题方法节点
- 应用场景节点
- 常见误区节点
- 节点之间的概念关系

直接在浏览器中打开 `index.html` 即可运行原型。

## 当前生成方式

当前版本支持两种生成方式：

### 1. LLM 生成

```text
输入词语
  -> 调用 OpenAI Responses API
  -> 使用 JSON Schema 约束输出
  -> 得到 nodes 和 edges
  -> 前端校验节点和关系
  -> 输出可视化图谱和 JSON 数据
```

### 2. 本地规则生成

如果没有输入 API Key，系统会使用本地规则生成器，保证页面仍然可以演示完整流程。

注意：当前是 GitHub Pages 静态网站，不能把 API Key 写入代码或仓库。页面中的 API Key 只用于本次浏览器会话。正式版应增加后端服务，由后端安全调用 LLM。

## 后续模块

1. 接入 LLM 或课程知识库，生成更具体的知识点。
2. 加入知识图谱质量校验，例如节点去重、关系合法性检查。
3. 再进入诊断问题生成模块。
4. 接入真实视频元数据和字幕文本。
5. 加入推荐评分：语义相似度、知识点匹配度、学习者水平。
