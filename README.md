# 教育视频知识图谱生成器

这是教育视频推荐系统的第一个模块，当前重点解决“根据用户输入词动态生成知识图谱”的问题。

当前版本已经不再读取预置领域图谱，而是根据用户输入的学习主题实时生成：

- 学习目标节点
- 前置知识节点
- 核心定义节点
- 表示方法节点
- 关键性质节点
- 解题方法节点
- 应用场景节点
- 常见误区节点
- 节点之间的概念关系

直接在浏览器中打开 `index.html` 即可运行原型。

## 当前生成方式

当前版本使用前端规则生成器：

```text
输入词语
  -> 生成学习目标节点
  -> 生成前置知识、核心定义、表示方法、性质、方法、应用、误区等节点
  -> 生成 prerequisite_of、has_definition、has_property 等关系
  -> 输出可视化图谱和 JSON 数据
```

这一步先解决知识图谱结构和动态生成流程。后续可以把规则生成器替换为 LLM 或知识库 API，让节点内容更具体、更准确。

## 后续模块

1. 接入 LLM 或课程知识库，生成更具体的知识点。
2. 加入知识图谱质量校验，例如节点去重、关系合法性检查。
3. 再进入诊断问题生成模块。
4. 接入真实视频元数据和字幕文本。
5. 加入推荐评分：语义相似度、知识点匹配度、学习者水平。
